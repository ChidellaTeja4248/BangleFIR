### Dataset Description
we have created an ornament dataset, to be specific, BangleFIR, a collection of high quality gold bangles, containing 4,379 pictures from 56 unique classes that are gathered from different web shopping destinations.

We have collected an ornament dataset of golden earrings. It contains 2,651 high-resolution golden earring images from different jewellery chain catalogues. These images are also used in online shopping apps.
